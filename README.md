# Customer Retention & Churn Analysis

## Objective
Analyze customer data to identify churn patterns, retention drivers, and customer lifetime trends for a subscription-based business.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Telco Customer Churn Dataset (7043 customers)

## Skills Applied
- Customer Retention Analysis
- Churn Analysis
- Data Cleaning
- Customer Lifetime Analysis
- Data Visualization
- Business Analytics

## Key Insights
- Approximately 26% of customers churned.
- Month-to-Month contracts experienced the highest churn rates.
- Customers with shorter tenure were more likely to leave.
- Higher monthly charges were associated with increased churn.
- Long-term contracts significantly improved customer retention.

## Recommendations
- Encourage customers to switch to annual contracts.
- Improve onboarding during the first 6 months.
- Offer loyalty rewards for long-term customers.
- Review pricing strategies for high-risk customer groups.
